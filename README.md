## ✨ Overview  

📷 **AI-Powered Posture Detection** built with **PoseNet + ml5.js**, running directly in your browser.  
Track workouts 🏋️, count reps ✅, measure calories 🔥, and get real-time feedback 🎯 — no installation required.
